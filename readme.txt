== Description ==

Easily upload and set a Favicon (browser icon, shortcut icon) for your WordPress site as well as an Apple Touch icon.

It is possible to set a different icon for both the front-end and dashboard areas.

== Installation ==

1. Upload the `quickfavicon` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

You can then click on `Favicon` on the administration menu to set up your Favicons or Apple Touch icon.
